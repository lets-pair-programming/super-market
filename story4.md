Story 4 - 促销活动
---

现在系统需要支持促销活动了。

有两种促销。\
P001 是买二送一活动。
如果披萨是买二送一商品，收据信息如下。
```json
{
  "data": [
    "****** SuperMarket receipt ******",
    "pizza: 4 x 15.00 --- 45.00",
    "milk: 3(L) x 12.30 --- 36.90",
    "---------------------------------",
    "Two-for-one:",
    "pizza: 1",
    "---------------------------------",
    "total: 81.90(CNY)",
    "       -15.00(CNY)",
    "*********************************"
  ]
}
```

P002是九五折促销。如果披萨是九五折商品，收据示例如下。
```json
{
  "data": [
    "****** SuperMarket receipt ******",
    "pizza: 4 x 15.00 --- 57.00 (-3.00)",
    "milk: 3(L) x 12.30 --- 36.90",
    "---------------------------------",
    "total: 93.90(CNY)",
    "       -3.00(CNY)",
    "*********************************"
  ]
}
```

如果一件商品既属于买二送一，又属于九五折促销，那么只有买二送一生效。
